おむつのうんち漏れ対策 - 横漏れ・背中漏れを防ぐコツ

公開日：2025年2月24日

おむつを替えようとしたら、うんちが背中まで漏れていた...。服もシーツも汚れて、洗濯が大変...。そんな経験、ありませんか？

特に新生児期のゆるゆるうんちは漏れやすく、多くのパパママが悩んでいます。この記事では、うんち漏れを防ぐための実践的な対策をご紹介します。

うんち漏れが起きる3つの原因

まず、なぜうんちが漏れるのか、原因を理解しましょう。

1. サイズが合っていない

最も多い原因がこれ。大きすぎても小さすぎても漏れます。

⚠️ サイズ選びの落とし穴

・大きすぎる → 隙間ができて漏れる

・小さすぎる → 圧迫されて漏れる

→ ジャストサイズが鉄則！

夜のおしっこ漏れ対策では「1サイズ大きめ」も選択肢ですが、うんち漏れに関してはジャストサイズが最重要です。

2. ギャザーが立っていない

おむつのギャザー（立体的なひだ）が寝たままだと、隙間ができて漏れやすくなります。

3. ゆるゆるうんち（新生児期）

新生児〜生後3ヶ月頃は、母乳やミルクの影響でうんちが水っぽく、漏れやすい時期です。

うんち漏れを防ぐ5つの対策

対策1：ジャストサイズを選ぶ

体重に合ったサイズを選びましょう。以下のサインが出たら、サイズアップのタイミングです。

  • 太ももに跡がつく
  • おへそが隠れない
  • テープが真ん中より外側で留まる
  • 頻繁に漏れるようになった

お子様に合ったおむつサイズを診断

体重を入力するだけで、最適なおむつサイズがわかります

診断ツールを使う

対策2：ギャザーをしっかり立てる

おむつを当てる時に、必ずギャザーを立てましょう。これだけで漏れがグッと減ります。

ギャザーの立て方

1. おむつを広げる

2. 足回りのギャザーを指でつまんで立てる

3. 背中側のギャザーも忘れずに

4. おむつを当てた後、もう一度確認

特に背中側のギャザーは見落としがち。仰向けに寝かせた時に、背中のギャザーがしっかり立っているか確認してください。

対策3：横漏れ防止機能があるおむつを選ぶ

最近のおむつは、横漏れ・背中漏れ対策が強化されているものがあります。特にゆるうんちの時期は、こういったおむつがおすすめです。

おすすめのおむつ

パンパース さらさらケア

立体ギャザーが優秀で、横漏れに強い。新生児期から人気。

→ 楽天で見る

メリーズ さらさらエアスルー

通気性が良く、ギャザーが柔らかい。肌に優しく漏れにくい。

→ 楽天で見る

ムーニー エアフィット

背中のギャザーが高めで、背中漏れ対策に優れている。

→ 楽天で見る

対策4：おむつの当て方を見直す

正しい当て方をマスターすれば、漏れは劇的に減ります。

正しいおむつの当て方

1. おむつの中心を赤ちゃんの背中の真ん中に合わせる

2. 背中側を少し高めに当てる（ウエストライン付近）

3. お腹側を引き上げる

4. テープは左右対称に、斜め下向きに留める

5. ギャザーを最終確認

特に重要：背中側を高めに

背中漏れの多くは、背中側のおむつが低すぎることが原因です。ウエストライン（腰骨あたり）まで来るように、少し高めに当てましょう。

対策5：うんちしたらすぐ交換

うんちは時間が経つとおむつの中で広がり、漏れやすくなります。においや様子で気づいたら、すぐに交換しましょう。

新生児期のゆるうんち対策

新生児〜生後3ヶ月頃は、うんちが水っぽく特に漏れやすい時期です。

この時期の特徴

  • 1日に5〜10回うんちすることも
  • 水っぽくて漏れやすい
  • 背中漏れが多い
  • 授乳のたびにうんちすることも

ゆるうんち期の対策

ゆるうんち対策のポイント

・新生児用の小さめサイズをジャストサイズで

・ギャザーは特に念入りに立てる

・背中側を高めに当てる

・こまめに確認して早めに交換

この時期は「漏れるのが普通」くらいに考えて、着替えを多めに用意しておくと気持ちが楽です。生後4ヶ月頃からうんちが固まってくるので、それまでの辛抱です。

それでも漏れる場合

対策をしても漏れる場合は、以下を試してみてください。

メーカーを変えてみる

おむつのメーカーによって、形やギャザーの位置が違います。今使っているおむつで漏れるなら、別のメーカーを試してみましょう。

体型に合ったおむつを探す

  • 細身の赤ちゃん → パンパース、メリーズ
  • ぽっちゃり赤ちゃん → ムーニー、グーン
  • 足が太め → ギャザーが柔らかいメリーズ

赤ちゃんの体型は様々なので、何種類か試して相性の良いものを見つけることが大切です。

まとめ

うんち漏れを防ぐポイントは以下の5つです。

  • ジャストサイズを選ぶ（大きすぎず小さすぎず）
  • ギャザーをしっかり立てる（特に背中側）
  • 横漏れ防止機能があるおむつを選ぶ
  • 背中側を高めに当てる
  • うんちしたらすぐ交換

特に新生児期は漏れやすい時期なので、「漏れて当たり前」くらいに気楽に構えましょう。生後4ヶ月を過ぎれば、うんちも固まって漏れにくくなります。

いろいろ試して、赤ちゃんに合ったおむつと当て方を見つけてくださいね。
